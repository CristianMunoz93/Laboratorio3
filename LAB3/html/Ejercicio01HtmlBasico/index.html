Respuestas de ejercicio

  1. Dar un par de ejemplos de uso de etiquetas meta.

    2. META http-equiv="X-UA-Compatible" content="IE=edge"
    META name="viewport" content="width=device-width, initial-scale=1.0"

  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML

    3. LANG=, XMLNS=

  3. Como se relaciona un header http con la etiqueta head de HTML?

    4. Mediante el atributo HTTP-EQUIV el cual provee información al header HTTP dependiendo el valor que tenga el atributo CONTENT

  4. Que es UTF8?

    5. Es el código de lenguaje español

  5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

    6. STYLE, es aplicable a casi todas las etiquetas HTML

  6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

    7. mediante el atributo TARGET="_BLANK"

  7. Explicar cuando usarían etiquetas semánticas y con que propósito.

  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.

    9. WIDTH,HEIGHT,BORDER entre otras. Para poder visualizar y distinguir con más facilidad las celdas de las tablas

  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    10. BODY en HTML tiene el contenido del documento HTML (etiquetas semánticas,estructura sintáctica del documento, etc)
    BODY en HTTP puede contener datos en cualquier formato(Ej:Datos ingresados en un formulario y archivos enviados en un proceso de upload)

  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

    11. COLSPAN, ROWSPAN. Se aplican a la etiqueta TD
